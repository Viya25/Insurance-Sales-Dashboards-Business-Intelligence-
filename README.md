Insurance & Sales Dashboards (Business Intelligence)
Files: Insurance_Dashboard.pbix/twbx, Sales Dashboard.xlsx, InsuranceData.csv

Project Overview: Created interactive visualizations to track Key Performance Indicators (KPIs) for the Insurance and Retail sectors.

Key Features:

  Insurance: Visualized claim ratios, policy distribution, and regional performance.

  Sales: Tracked monthly revenue growth, profit margins, and customer segmentation.

  Tools: Leveraged DAX (Power BI) or Calculated Fields (Tableau) for dynamic reporting.
